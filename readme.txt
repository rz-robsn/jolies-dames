========				
	Run Instructions		
========		

	Either build the solution yourself buy opening the brown-skin-game.sln file with Visual Studio or
	just execute the file Debug/brown-skin-game.exe

========
	Controls 
========

 F1				Resets the camera and light 
 F2				Toggles the position of the light 
 F3 and F4 		Changes the position of the camera
 F5				Select/Move a piece
 F6				New Game
 F12			Quit
